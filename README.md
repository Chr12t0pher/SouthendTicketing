# SouthendTicketing

Gets ticket information on upcoming Southend games.

## Installation
```
$ git clone https://github.com/Chr12t0pher/SouthendTicketing.git
$ cd SouthendTicketing
$ pip install -r requirements.txt
$ python wsgi.py
```
